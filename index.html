GrafKom 2D — Dokumentasi
Grafika Komputer · Kelompok 1
01 — Tentang Proyek

Proyek akhir mata kuliah Grafika Komputer yang mengimplementasikan berbagai algoritma fundamental grafika 2D secara interaktif berbasis web menggunakan HTML5 Canvas API dan JavaScript murni.

Rendering dilakukan secara pixel-by-pixel tanpa memanfaatkan fungsi drawing bawaan Canvas (kecuali fillRect untuk plot piksel).

03 — Tipe Garis
  • Solid — semua piksel aktif
  • Dashed — pola 6 ON, 4 OFF
  • Dotted — pola 1 ON, 3 OFF
  • Dash-Dot — pola 6 ON, 3 OFF, 1 ON, 3 OFF
04 — Transformasi 2D

Koordinat homogen 3×3: M = T · R · S · Sh

TransformasiParameter
Translasi (T)tx, ty dalam piksel
Rotasi (R)sudut derajat, pusat objek
Skala (S)0.1× – 4.0×
Shear (Sh)−200 s/d +200
RefleksiX, Y, Origin, y=x
05 — Cara Penggunaan
  • Pilih alat dari toolbar, lalu klik/drag di canvas.
  • Poligon: atur jumlah sisi, lalu klik titik-titik. Otomatis selesai.
  • Flood Fill: klik area terbuka di canvas.
  • Pensil: tahan dan gerak mouse.
  • Penghapus: tahan dan gerak mouse.
  • Shortcut: V=Cursor, P=Pensil, E=Hapus, F=Fill, L=Garis, C=Lingkaran
